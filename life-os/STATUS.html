⚠️ Access Issue

The servers are running inside a Docker container and are only accessible via localhost within that container.

To access Life OS, choose one option:

  1. Port Forwarding - If you have SSH access to the host: 
    ssh -L 3000:localhost:3000 -L 8000:localhost:8000 your-host
  2. Deploy to Cloud - Deploy to Railway (backend) + Vercel (frontend):
    • Push code to GitHub
    • Connect Railway to your repo (backend)
    • Connect Vercel to your repo (frontend)
  3. Docker Port Mapping - Restart OpenClaw container with port mappings: 
    docker run -p 3000:3000 -p 8000:8000 ...
